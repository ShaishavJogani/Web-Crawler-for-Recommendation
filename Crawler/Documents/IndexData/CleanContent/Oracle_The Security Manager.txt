A security manager object define security policy application . This policy specify action unsafe sensitive . Any action allow security policy cause SecurityException throw . An application also query security manager discover action allow . Typically , web applet run security manager provide browser Java Web Start plugin . Other kind application normally run without security manager , unless application define one . If security manager present , application security policy act without restriction . This section explain application interact exist security manager . For detailed information , include information design security manager , refer Security Guide . Interacting Security Manager The security manager object type SecurityManager ; obtain reference object , invoke System.getSecurityManager . SecurityManager appsm = System.getSecurityManager ( ) ; If security manager , method return null . Once application reference security manager object , request permission specific thing . Many class standard libraries . For example , System.exit , terminate Java virtual machine exit status , invoke SecurityManager.checkExit ensure current thread permission shut application . The SecurityManager class defines many method use verify kind operation . For example , SecurityManager.checkAccess verifies thread access , SecurityManager.checkPropertyAccess verifies access specify property . Each operation group operation checkXXX ( ) method . In addition , set checkXXX ( ) method represent set operation already subject protection security manager . Typically , application directly invoke checkXXX ( ) method . Recognizing Security Violation Many action routine without security manager throw SecurityException run security manager . This true even invoke method n't document throw SecurityException . For example , consider following code use read file : reader = new FileReader ( `` xanadu.txt '' ) ; In absence security manager , statement execute without error , provide xanadu.txt exists readable . But suppose statement insert web applet , typically run security manager allow file input .
