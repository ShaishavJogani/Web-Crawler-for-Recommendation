This lesson introduce fundamental concept JMX API , namely manage bean , MBeans . An MBean managed Java object , similar JavaBeans component , follow design pattern set forth JMX specification . An MBean represent device , application , resource need manage . MBeans expose management interface consist following : A set readable writable attribute , . A set invokable operation . A self-description . The management interface change throughout life MBean instance . MBeans also emit notification certain predefined event occur . The JMX specification defines five type MBean : Standard MBeans Dynamic MBeans Open MBeans Model MBeans MXBeans The example trail demonstrate simple type MBean , namely standard MBeans MXBeans .
