Upon instantiation , JTable object pass table model object manage data display . By default , JTable object inherit DefaultTable object custom TableModel object specify , default , model manage string . To handle object , perform calculation , retrieve data database program , must design custom TableModel object , implement TableModel interface . See Creating Table Model detail . To detect change data manage table model object , JTable class need implement TableModelListener interface , call addTableModelListener ( ) catch event , override tableChanged ( ) respond listener event . See Listening Data Changes detail . The Table Model Listener API The TableModelListener Interface Because TableModelListener one method , corresponding adapter class .
