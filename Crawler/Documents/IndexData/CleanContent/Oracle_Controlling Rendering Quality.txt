Use Graphics2D class render hint attribute specify whether want object render quickly possible whether prefer rendering quality high possible . To set change rendering hint attribute Graphics2D context , construct RenderingHints object pas Graphics2D use setRenderingHints method . If want set one hint , call Graphics2D setRenderingHint specify key-value pair hint want set . ( The key-value pair define RenderingHints class . ) For example , set preference antialiasing use possible , could use setRenderingHint : public void paint ( graphic g ) { Graphics2D g2 = ( Graphics2D ) g ; RenderingHints rh = new RenderingHints ( RenderingHints.KEY_TEXT_ANTIALIASING , RenderingHints.VALUE_TEXT_ANTIALIAS_ON ) ; g2.setRenderingHints ( rh ) ; ... } Note : Not platform support modification rendering mode , specify render hint guarantee use .
