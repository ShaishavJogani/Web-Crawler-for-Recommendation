Defining Fancy Line Styles Fill Patterns Using Java 2D Stroke Paint class , define fancy line style fill pattern . Line Styles Line style define stroke attribute Graphics2D render context . To set stroke attribute , create BasicStroke object pas Graphics2D setStroke method . A BasicStroke object hold information line width , join style , end-cap style , dash style . This information use Shape render draw method . The line width thickness line measure perpendicular trajectory . The line width specify float value user coordinate unit , roughly equivalent 1/72 inch default transform use . The join style decoration apply two line segment meet . BasicStroke support following three join style : JOIN_BEVEL JOIN_MITER JOIN_ROUND The end-cap style decoration apply line segment end . BasicStroke support following three end-cap style : CAP_BUTT CAP_ROUND CAP_SQUARE The dash style define pattern opaque transparent section apply along length line . The dash style define dash array dash phase . The dash array define dash pattern . Alternating element array represent dash length length space dash user coordinate unit . Element 0 represent first dash , element 1 first space , . The dash phase offset dash pattern , also specify user coordinate unit . The dash phase indicate part dash pattern apply beginning line .
