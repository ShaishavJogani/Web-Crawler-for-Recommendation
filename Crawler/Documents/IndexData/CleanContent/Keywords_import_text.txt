importis Java keyword . It declare Java class use code import statement . Once Java class declare , class name use code without specify package class belong . Use '* ' character declare class belong package . Syntax : The static import construct allow unqualified access static member without inherit type contain static member : import static java.lang.Math.PI ; Once static member import , may use without qualification : double r = co ( PI * theta ) ; Caveat : use static import sparingly avoid pollute program 's namespace !
