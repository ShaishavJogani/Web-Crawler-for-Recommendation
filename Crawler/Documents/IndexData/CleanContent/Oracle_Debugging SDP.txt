You enable debug message use -Dcom.sun.sdp.debug flag . If specify file , message output file . Otherwise , message print standard output . This first example show sample message print standard output : % java -Dcom.sun.sdp.conf=sdp.conf -Dcom.sun.sdp.debug ExampleApplicaton BIND 192.0.2.1:5000 ( socket convert SDP protocol ) CONNECT 129.156.232.160:80 ( match ) CONNECT 192.0.2.2:80 ( socket convert SDP protocol ) This second example show output redirect file name debug.log : % java -Dcom.sun.sdp.conf=sdp.conf -Dcom.sun.sdp.debug=debug.log ExampleApplication [ 1 ] 27310 % tail -f debug.log BIND 192.0.2.1:5000 ( socket convert SDP protocol )
