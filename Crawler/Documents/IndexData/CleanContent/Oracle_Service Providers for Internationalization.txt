Service provider internationalization enable plug-in locale-dependent data service . Because locale-dependent data service plugged-in , third party able provide implementation locale-sensitive class java.text java.util package . A service set program interface class provide access specific application 's functionality feature . A service provider interface ( SPI ) set public interface abstract class service defines . A service provider implement SPI . Service provider enable create extensible application , extend without modify original code base . You enhance functionality new plug-in module . For information service provider extensible application , see Creating Extensible Applications .
