You may need include package version information JAR file 's manifest . You provide information follow header manifest : Headers manifest Header Definition Name The name specification . Specification-Title The title specification . Specification-Version The version specification . Specification-Vendor The vendor specification . Implementation-Title The title implementation . Implementation-Version The build number implementation . Implementation-Vendor The vendor implementation . One set header assign package . The versioning header appear directly beneath Name header package . This example show versioning header : Name : java/util/ Specification-Title : Java Utility Classes Specification-Version : 1.2 Specification-Vendor : Example Tech , Inc. Implementation-Title : java.util Implementation-Version : build57 Implementation-Vendor : Example Tech , Inc. For information package version header , see Package Versioning specification .
