You already learn draw shape represent java.awt.geom package . To create complicated geometry , polygon , polylines , star use another class package , GeneralPath . This class implement Shape interface represent geometric path construct line , quadratic cubic curve . The three constructor class create GeneralPath object default winding rule ( WIND_NON_ZERO ) , give winding rule ( WIND_NON_ZERO WIND_EVEN_ODD ) , specified initial coordinate capacity . The winding rule specify interior path determine . public void paint ( Graphics g ) { Graphics2D g2 = ( Graphics2D ) g ; ... } To create empty GeneralPath instance call new GeneralPath ( ) add segment shape use following method : moveTo ( float x , float ) – Moves current point path give point lineTo ( float x , float ) – Adds line segment current path quadTo ( float ctrlx , float ctrly , float x2 , floaty2 ) – Adds quadratic curve segment current path curveTo ( float ctrlx1 , float ctrly1 , float ctrlx2 , float ctrly2 , float x3 , floaty3 ) – Adds cubic curve segment current path closePath ( ) – Closes current path The following example illustrate draw polyline use GeneralPath : // draw GeneralPath ( polyline ) int x2Points [ ] = { 0 , 100 , 0 , 100 } ; int y2Points [ ] = { 0 , 50 , 50 , 0 } ; GeneralPath polyline = new GeneralPath ( GeneralPath.WIND_EVEN_ODD , x2Points.length ) ; polyline.moveTo ( x2Points [ 0 ] , y2Points [ 0 ] ) ; ( int index = 1 ; index < x2Points.length ; index++ ) { polyline.lineTo ( x2Points [ index ] , y2Points [ index ] ) ; } ; g2.draw ( polyline ) ; This example illustrate draw polygon use GeneralPath : // draw GeneralPath ( polygon ) int x1Points [ ] = { 0 , 100 , 0 , 100 } ; int y1Points [ ] = { 0 , 50 , 50 , 0 } ; GeneralPath polygon = new GeneralPath ( GeneralPath.WIND_EVEN_ODD , x1Points.length ) ; polygon.moveTo ( x1Points [ 0 ] , y1Points [ 0 ] ) ; ( int index = 1 ; index < x1Points.length ; index++ ) { polygon.lineTo ( x1Points [ index ] , y1Points [ index ] ) ; } ; polygon.closePath ( ) ; g2.draw ( polygon ) ; Note difference two last code example closePath ( ) method . This method make polygon polyline draw straight line back coordinate last moveTo . To add specific path end GeneralPath object use one append ( ) method .
