Applets Packaged JAR Files To start applet HTML file run inside browser , use applet tag . For information , see Java Applets lesson . If applet bundle JAR file , thing need differently use archive parameter specify relative path JAR file . As example , use TicTacToe demo applet . The applet tag HTML file display applet mark like : < applet code=TicTacToe.class width= '' 120 '' height= '' 120 '' > < /applet > If TicTacToe demo package JAR file name TicTacToe.jar , modify applet tag addition archive parameter : < applet code=TicTacToe.class archive= '' TicTacToe.jar '' width= '' 120 '' height= '' 120 '' > < /applet > The archive parameter specify relative path JAR file contain TicTacToe.class . For example assume JAR file HTML file directory . If , must include JAR file 's relative path archive parameter 's value . For example , JAR file one directory HTML file directory call applet , applet tag would look like : < applet code=TicTacToe.class archive= '' applets/TicTacToe.jar '' width= '' 120 '' height= '' 120 '' > < /applet > JAR Files Applications You run JAR package application Java launcher ( java command ) . The basic command : java -jar jar-file The -jar flag tell launcher application package JAR file format . You specify one JAR file , must contain application-specific code . Before execute command , make sure runtime environment information class within JAR file application 's entry point . To indicate class application 's entry point , must add Main-Class header JAR file 's manifest . The header take form : Main-Class : classname The header 's value , classname , name class application 's entry point . For information , see Setting Application 's Entry Point section . When Main-Class set manifest file , run application command line : java -jar app.jar To run application JAR file another directory , must specify path directory : java -jar path/app.jar
