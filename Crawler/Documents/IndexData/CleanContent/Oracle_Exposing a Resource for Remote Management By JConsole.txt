This example require version 6 Java SE platform . To monitor Main JMX agent remotely , follow step : If already , save jmx_examples.zip work_dir directory . Unzip bundle sample class use following command terminal window . unzip jmx_examples.zip Compile example Java class within work_dir directory . javac com/example/*.java Start Main application , specify property expose Main remote management . ( For Windows , use carat ( ^ ) instead backslash ( \ ) break long command multiple line ) : java -Dcom.sun.management.jmxremote.port=9999 \ -Dcom.sun.management.jmxremote.authenticate=false \ -Dcom.sun.management.jmxremote.ssl=false \ com.example.Main A confirmation Main wait something happen generate . Select Remote Process , type following Remote Process field : hostname:9999 In address , hostname name remote machine Main application run 9999 number port out-of-the-box JMX connector connect . Click Connect . A summary current activity Java Virtual Machine ( Java VM ) Main run display . Click MBeans tab . This panel show MBeans currently register remote MBean server .
