IPv4 IPv6 incompatibility Internet Protocol version 4 ( IPv4 ) long industry standard version Internet Protocol ( IP ) deliver data Internet . Internet Protocol version 6 ( IPv6 ) next generation Internet layer protocol . Both version IP use today . IPv4 address 32-bits long , write decimal format , separate period . IPv6 address 128-bits long , write hexadecimal format , separate colon . IPv4 address use IPv6 , IPv6 support special class address : IPv4-mapped address . In IPv4-mapped address , first 80 bit set zero , next 16 bit set 1 , last 32 bit represent IPv4 address . For example , IP address express format : IPv4 address IPv4-mapped address ( use IPv6 ) 192.0.2.1 : :ffff:192.0.2.1 By default , IPv6 enable IB adaptor , Java platform use IPv6 . However , IPv4-mapped address currently available Solaris OS Linux .
