The system tray specialized area desktop user access currently run program . This area may refer differently various operate system . On Microsoft Windows , system tray refer Taskbar Status Area , GNU Network Object Model Environment ( GNOME ) Desktop refer Notification Area . On K Desktop Environment ( KDE ) area refer System Tray . However , system tray area share application run desktop . The java.awt.SystemTray class introduce Java™ SE version 6 represent system tray desktop . The system tray access call static SystemTray.getSystemTray ( ) method . Before call method , use static method isSupported ( ) check system tray support . If system tray support platform , isSupported ( ) method return false . If application attempt call getSystemTray ( ) method case , method throw java.lang.UnsupportedOperationException . An application create instance SystemTray class . Only single instance create within class exist , instance obtain use getSystemTray ( ) method . The system tray contain one tray icon add tray use add ( java.awt.TrayIcon ) method . They remove longer need remove ( java.awt.TrayIcon ) method . Note : The add ( ) method throw AWTException operating system Java runtime determine icon add system tray . For example , AWTException throw X-Window desktop system tray exist desktop . The TrayIcon class functionality go beyond icon display tray . It also include text tooltip , pop-up menu , ballon message , set listener associate . A TrayIcon object generate various mouse event support addition correspond listener receive notification event . The TrayIcon class process event . For example , default , right-click perform tray icon , display specified pop-up menu . When double-click perform , TrayIcon object generate ActionEvent launch application . When mouse pointer hovers tray icon , tooltip display . The order icon display tray specify — platform- implementation-dependent . getSystemTray Gets SystemTray instance represent desktop 's tray area . This method always return instance per application . On platform system tray may support . Use isSupported ( ) method check system tray support . isSupported Returns information whether system tray support current platform . In addition display tray icon , minimal system tray support include either pop-up menu ( see TrayIcon.setPopupMenu ( PopupMenu ) method ) action event ( see TrayIcon.addActionListener ( ActionListener ) ) .
