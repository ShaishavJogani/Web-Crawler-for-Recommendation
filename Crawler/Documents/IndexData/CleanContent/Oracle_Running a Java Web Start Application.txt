Users run Java Web Start application following way : Running Java Web Start Application From Browser Running Java Web Start Application From Java Cache Viewer Running Java Web Start Application From Desktop Note : To run application deploy Java Web Start technology , must compatible version Java Runtime Environment ( JRE ) software . The complete Java Java Development Kit ( JDK ) require . Running Java Web Start Application From Browser You run Java Web Start application browser click link application 's JNLP file . The following text example link JNLP file . < href= '' /some/path/Notepad.jnlp '' > Launch Notepad Application < /a > Java Web Start software load run application base instruction JNLP file . Try : Run Notepad Running Java Web Start Application From Java Cache Viewer If use least Java Platform , Standard Edition 6 later , run Java Web Start application Java Cache Viewer . When Java Web Start software first load application , information application 's JNLP file store local Java Cache Viewer . To launch application , need return web page first launch ; launch Java Cache Viewer . To open Java Cache Viewer : Open Control Panel . Double click Java icon . The Java Control Panel open . Select General tab . Click View . The Java Cache Viewer open . The application list Java Cache Viewer screen .
