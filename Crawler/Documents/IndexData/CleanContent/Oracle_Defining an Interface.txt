An interface declaration consist modifier , keyword interface , interface name , comma-separated list parent interface ( ) , interface body . For example : public interface GroupedInterface extend Interface1 , Interface2 , Interface3 { // constant declaration // base natural logarithm double E = 2.718282 ; // method signature void doSomething ( int , double x ) ; int doSomethingElse ( String ) ; } The public access specifier indicate interface use class package . If specify interface public , interface accessible class define package interface . An interface extend interface , class subclass extend another class . However , whereas class extend one class , interface extend number interface . The interface declaration include comma-separated list interface extend . The Interface Body The interface body contain abstract method , default method , static method . An abstract method within interface follow semicolon , brace ( abstract method contain implementation ) . Default method define default modifier , static method static keyword . All abstract , default , static method interface implicitly public , omit public modifier . In addition , interface contain constant declaration . All constant value define interface implicitly public , static , final . Once , omit modifier .
