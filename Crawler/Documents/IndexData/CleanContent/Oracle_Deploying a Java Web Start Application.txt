You deploy Java Web Start application use createWebStartLaunchButton function Deployment Toolkit script . Java Web Start application launch use Java Network Launch Protocol ( JNLP ) The createWebStartLaunchButton function generate link ( HTML anchor tag - < > ) Java Web Start application 's JNLP file . This generate anchor tag Java Web Start application 's button . When end user click Launch button , Deployment Toolkit script ensure appropriate Java Runtime Environment ( JRE ) software instal launch Java Web Start application . Note : Depending type browser , might unable view HTML generate Deployment Toolkit script try view source web page . To view generated HTML , try save HTML page load , use tool Firebug ( Mozilla Firefox add-on ) .Note : If client required minimum version JRE software , Deployment Toolkit script redirect browser http : //www.java.com allow user download late JRE software .
