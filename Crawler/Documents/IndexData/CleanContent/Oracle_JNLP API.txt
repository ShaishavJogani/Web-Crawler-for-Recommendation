Rich Internet application ( RIAs ) use Java Network Launch Protocol ( JNLP ) API perform extensive operation user 's environment . When launch use JNLP , even unsigned RIAs perform following operation user 's permission : They use FileOpenService FileSaveService API access user 's file system.. They use ClipboardService API access share system-wide clipboard . They use PrintService API access printing function . They use PersistenceService API access persistence storage . They use DownloadService API control RIA download cache . They use DownloadServiceListener API determine progress RIA 's download . They use SingleInstanceService API decide handle argument multiple instance RIA launch . They use ExtendedService API request permission open certain file open . Check JNLP API documentation see complete list functionality available RIAs launch use JNLP .
