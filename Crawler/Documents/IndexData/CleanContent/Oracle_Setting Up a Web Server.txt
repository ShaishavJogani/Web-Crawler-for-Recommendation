You might need configure web server handle Java Network Launch Protocol ( JNLP ) file . If web server set properly , Java Web Start application launch click link JNLP file . Configure web server file .jnlp extension set application/x-java-jnlp-file MIME type . The specific step set JNLP MIME type vary depend web server . As example , configure Apache web server , add following line mime.types file . application/x-java-jnlp-file JNLP For web server , check documentation instruction set MIME type .
