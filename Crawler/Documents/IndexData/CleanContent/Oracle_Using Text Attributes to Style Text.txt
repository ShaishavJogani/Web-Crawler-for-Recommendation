Applications typically need capability apply following text attribute : Underline – A line draw underneath text Strikethrough – A horizontal line draw text Superscript Subscript – A text letter appear slightly line correspondingly line Kerning – The adjustment space character These text attribute apply use Java 2D TextAttribute class . To apply text attribute add Font object . For example : Map < TextAttribute , Object > map = new Hashtable < TextAttribute , Object > ( ) ; map.put ( TextAttribute.KERNING , TextAttribute.KERNING_ON ) ; font = font.deriveFont ( map ) ; graphics.setFont ( font ) ; The code example represent show application text attribute following order : Sample string ( text attribute apply ) Kerning Kerning Underlining Kerning , Underlining Strikethrough Kerning , Underlining , Strikethrough Color Note : If n't see applet running , need install least Java SE Development Kit ( JDK ) 7 release . The complete code applet AttributedText.java .
