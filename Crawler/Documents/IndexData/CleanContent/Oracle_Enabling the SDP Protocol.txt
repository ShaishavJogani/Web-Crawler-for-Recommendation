SDP support disable default . To enable SDP support , set com.sun.sdp.conf system property provide location configuration file . The following example start application use configuration file name sdp.conf : % java -Dcom.sun.sdp.conf=sdp.conf -Djava.net.preferIPv4Stack=true ExampleApplication ExampleApplication refers client application attempt connect IB adaptor . Note example specify another system property , java.net.preferIPv4Stack . See Issues section information property use .
