Use getResource method read resource JAR file . For example , following code retrieves image JAR file . // Get current classloader ClassLoader cl = this.getClass ( ) .getClassLoader ( ) ; // Create icon Icon saveIcon = new ImageIcon ( cl.getResource ( `` images/save.gif '' ) ) ; Icon cutIcon = new ImageIcon ( cl.getResource ( `` images/cut.gif '' ) ) ; The example assume following entry exist application 's JAR file : images/save.gif images/cut.gif
