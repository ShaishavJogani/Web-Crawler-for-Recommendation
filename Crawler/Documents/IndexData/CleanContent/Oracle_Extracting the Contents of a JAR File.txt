The basic command use extract content JAR file : jar xf jar-file [ archived-file ( ) ] Let 's look option argument command : The x option indicate want extract file JAR archive . The f option indicate JAR file file extract specify command line , rather stdin . The jar-file argument filename ( path filename ) JAR file extract file . archived-file ( ) optional argument consist space-separated list file extract archive . If argument present , Jar tool extract file archive . As usual , order x f option appear command n't matter , must space . When extract file , Jar tool make copy desired file write current directory , reproduce directory structure file archive . The original JAR file remain unchanged .
