The Jar tool provide u option use update content exist JAR file modify manifest add file . The basic command add file format : jar uf jar-file input-file ( ) In command : The u option indicate want update exist JAR file . The f option indicate JAR file update specify command line . jar-file exist JAR file update . input-file ( ) space-delimited list one file want add JAR file . Any file already archive pathname file add overwrite . When create new JAR file , optionally use -C option indicate change directory . For information , see Creating JAR File section .
