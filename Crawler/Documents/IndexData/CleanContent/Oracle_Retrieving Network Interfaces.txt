The NetworkInterface class public constructor . Therefore , create new instance class new operator . Instead , following static method available retrieve interface detail system : getByInetAddress ( ) , getByName ( ) , getNetworkInterfaces ( ) . The first two method use already know IP address name particular interface . The third method , getNetworkInterfaces ( ) return complete list interface machine . Network interface hierarchically organize . The NetworkInterface class include two method , getParent ( ) getSubInterfaces ( ) , pertinent network interface hierarchy . The getParent ( ) method return parent NetworkInterface interface . If network interface subinterface , getParent ( ) return non-null value . The getSubInterfaces ( ) method return subinterfaces network interface .
