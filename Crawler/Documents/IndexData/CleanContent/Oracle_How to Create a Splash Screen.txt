Almost modern application splash screen . Typically splash screen use following purpose : Advertising product Indicating user application launch long startup time Providing information need per visit Java Foundation Classes , Swing Abstract Windowing Toolkit ( AWT ) , enable developer create splash screen Java technology application . However , main purpose splash screen provide user feedback application 's startup , delay application 's startup moment splash screen pop minimal . Before splash screen pop , application load initialize Java™ Virtual Machine ( JVM ) , AWT , Swing , sometimes application-dependent library well . The resulting delay several second make use Java™ technology-based splash screen less desirable . Fortunately , Java™ SE 6 provide solution allow application display splash screen much earlier , even virtual machine start . A Java application launcher able decode image display simple non-decorated window . The splash screen display gif , png , jpeg image , transparency , translucency , animation .
